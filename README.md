# Seen Transaction

## Requirements

- Node.js >= 22.0.0

## Installation

1. Clone the repo.
2. Run `npm install`.
3. Copy `.env.example` to `.env`:

   ```sh
   cp .env.example .env
   ```

## Running

- `npm start`: Run in development mode.

## Test

- `npm run test`: Run all tests.
- `npm run test:unit`: Run unit tests.
- `npm run test:integration`: Run integration tests.
